🦀 Create the crab'd GitHub App

This opens GitHub's New GitHub App form, pre-filled with crab'd's permissions and events (webhook disabled). Nothing is sent anywhere but GitHub — you own the App and its key.

Then, on GitHub

  1. Review the form and click Create GitHub App. If the name is taken, pick a unique one (e.g. crabd-yourname).
  2. On the App's page, note the App ID and click Generate a private key (downloads a .pem).
  3. Open Install App and install it on the repos crab'd should act on.
  4. Store CRABD_APP_ID and CRABD_APP_PRIVATE_KEY as repo/org secrets.
